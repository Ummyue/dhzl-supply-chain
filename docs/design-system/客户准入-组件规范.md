## 一、Button 按钮

### 1.1 变体规范
| 变体 | 用法 | 类比 AntD | Tailwind |
|---|---|---|---|
| **Primary** | 主操作（提交 / 保存 / 下一步） | `<Button type="primary">` | `bg-blue-600 text-white` |
| **Default** | 次操作（取消 / 返回） | `<Button>` | `bg-white border-slate-300` |
| **Text** | 弱操作（更多 / 链接） | `<Button type="text">` | `text-blue-600` |
| **Danger** | 危险操作（删除 / 驳回） | `<Button danger>` | `bg-rose-500 text-white` |
| **Link** | 行内操作 | `<Button type="link">` | `text-blue-600 hover:underline` |

### 1.2 尺寸
| 尺寸 | 高 px | 用法 |
|---|---|---|
| SM | 24 | 表格行内 / 紧凑区 |
| MD (default) | 32 | **默认**（90% 场景） |
| LG | 40 | 弹窗主操作 / 重要 CTA |

### 1.3 圆角
统一 6px（Tailwind `rounded-md` / AntD `borderRadius`）

### 1.4 在客户准入中的具体应用
| 页面 | 按钮 | 类型 |
|---|---|---|
| 准入申请列表 | 顶部"+ 准入申请" | Primary MD |
| | 重置 / 查询 | Default MD / Primary MD |
| 新增准入申请 | 提交（底部 fixed） | Primary LG |
| | 取消（底部 fixed） | Default LG |
| 详情页 | 返回准入申请 | Text SM |
| | 信息有误？申请修改 | Link SM（蓝） |
| 企业信息 | 申请修改 | Primary MD |

## 二、Input 输入框

### 2.1 变体规范
| 状态 | 类比 AntD | 视觉 |
|---|---|---|
| Default | `<Input>` | `border: #d9d9d9 / bg: #fff` |
| Hover | — | `border: #4096ff` |
| Focus | — | `border: #1677ff` + 2px 外阴影 |
| Disabled | `<Input disabled>` | `bg: #f5f5f5 / text: #00000040` |
| Readonly | `<Input readOnly>` | `bg: #fafafa / text: #00000073` |
| Error | `<Input status="error">` | `border: #f5222d` |

### 2.2 高度
- 默认 32px（`h-8`）
- 大尺寸 40px（`h-10`，用于底部 fixed 提交按钮所在的 form）
- 圆角 6px

### 2.3 在客户准入中的具体应用
- **必填字段**：label 加红色 `*`（CSS `::before` 伪元素）
- **自动带入只读**（v1.7.37）：前 8 字段 + 法人 5 字段
  - `bg: #f1f5f9 / text: #64748b / cursor: not-allowed`
- **可编辑字段**：白底 + 边框

## 四、Table 表格

### 4.1 基础规范
| 元素 | 样式 |
|---|---|
| 表头 | `bg: #fafafa / color: #000000d9 / fontSizeSM 13px / fontWeightMedium 500` |
| 行高 | 48px（默认）/ 40px（紧凑）/ 32px（极小） |
| 行边框 | 1px `#f0f0f0` 横向 |
| 行 hover | `bg: #fafafa` |
| 数字列 | 右对齐 + `font-variant-numeric: tabular-nums` |

### 4.2 在客户准入中的具体应用

  - 表头 3 列（科目 + 2 个期间）
  - 加粗行：流动资产合计 / 资产总计 / 流动负债合计 / 负债合计 / 所有者权益 / 负债及权益合计

## 八、Form 表单

### 8.1 必填标识
```html
<label class="req">企业名称</label>  <!-- ::before { content: '*'; color: #f5222d; } -->
```

### 8.2 label 布局
- 桌面端：label 右对齐 160px 宽 / 30px 行高
- 平板端：label 顶部 / 100% 宽

### 8.3 字段网格
| 屏幕 | 网格 |
|---|---|
| 桌面 1440px | 3 列（160px label + 1fr + 160px label + 1fr + 160px label + 1fr） |
| 平板 768px | 1 列（label 顶部 + input 100% 宽） |

### 8.4 校验反馈
- 通过：边框变绿
- 失败：边框变红 + 下方红色错误文字（12px）
- 必填漏填：toast 提示"请补全必填项：XXX"

## 十五、命名约定

| 类别 | 命名 | 例 |
|---|---|---|
| 颜色 | `{用途}` | `colorPrimary` / `colorSuccess` |
| 字号 | `fontSize{XXX}` | `fontSizeMD` = 14px |
| 间距 | `space{XXX}` | `spaceLG` = 16px |
| 圆角 | `borderRadius{XXX}` | `borderRadiusLG` = 8px |
| 阴影 | `boxShadow{XXX}` | `boxShadowSM` |
| 组件 | `{组件}{变体}` | `ButtonPrimary` / `TagSuccess` |

---

## 十六、可访问性（A11y）

- 所有交互元素有 focus 态（蓝色边框 + 2px 外阴影）
- 必填项不仅靠 `*` 颜色，还配 `aria-required="true"`
- 表格行 hover 同时显示 tooltip 提示
- 颜色对比度 ≥ 4.5:1（WCAG AA）
- 按钮/链接 `title` 必填
